# Finance Analysis Skill

This skill allows the agent to perform in-depth analysis of financial statements, market reports, and stock performance.

## 🛠️ Components

- **Fundamental Analysis**: Examining balance sheets, P&L, and cashflow.
- **Technical Indicators**: Calculating moving averages, RSI, and Bollinger Bands.
- **Sentiment Tracking**: Analyzing news headlines for market sentiment.

## 📈 Capabilities

The skill includes:
- **Financial Ratios Calculation**: Evaluating company health using key metrics (P/E, ROE, etc.).
- **Sector Peer Comparison**: Benchmarking a company against its competitors.
- **Trend Detection**: Identifying long-term and short-term trends in financial datasets.

## 📡 Requirements

- Standard HTTP headers for API requests.
- Reliable caching for historical datasets.
- Pre-processing for unstructured financial data.
