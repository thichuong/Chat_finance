# Chat Finance Frontend (Vanilla JS)

This is a pure HTML, CSS, and Vanilla JavaScript implementation of the Chat Finance AI interface.

## No Build Step Required
This frontend does not require `npm`, `node_modules`, or any build tools (Vite, Webpack, etc.). 
The backend (`api.py`) serves the static files directly from this directory.

## File Structure
- `index.html`: Main HTML structure.
- `style.css`: Premium design system, layout, and animations.
- `app.js`: Application logic (market data, chat streaming, UI updates).

## Dependencies (via CDN)
- [Lucide Icons](https://lucide.dev/): SVG icons.
- [Marked.js](https://marked.js.org/): Markdown parsing for chat messages.
- [TradingView](https://www.tradingview.com/): Financial charts.
